### 4.3 自定义评测脚本

```python
# benchmark_framework.py
import json
import time
from dataclasses import dataclass

@dataclass
class BenchmarkResult:
    tool: str
    task: str
    tokens_in: int
    tokens_out: int
    compression_ratio: float
    passed: bool
    time_ms: float

def run_benchmark(tool, tasks, model):
    results = []
    for task in tasks:
        result = execute_task(tool, task, model)
        results.append(result)
    return results

def analyze_results(results):
    return {
        'avg_compression': sum(r.compression_ratio for r in results) / len(results),
        'pass_rate': sum(r.passed for r in results) / len(results),
        'avg_time': sum(r.time_ms for r in results) / len(results)
    }
```

## 七、快速开始

### 7.1 环境准备

```bash
# 1. 安装Docker
# 2. 安装Python 3.10+
# 3. 安装测试工具
pip install swebench terminal-bench

# 4. 准备测试任务
mkdir benchmark_tasks
```

### 7.2 运行基准测试

```bash
# 1. 基线测试 (无压缩工具)
python run_baseline.py --tasks benchmark_tasks

# 2. 安装测试工具
cargo install rtk lean-ctx

# 3. 对比测试
python run_comparison.py --tools rtk lean-ctx --tasks benchmark_tasks

# 4. 生成报告
python generate_report.py --results results/
```
