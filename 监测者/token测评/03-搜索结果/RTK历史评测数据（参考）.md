## 一、项目概述

RTK 是基于 Rust 开发的轻量级 CLI 代理工具，专为 Claude Code 等 AI 编程工具设计。通过"做减法"的方式，净化终端输出内容，节省 Token 消耗。

**核心理念**: 不改代码、不侵流程、不毁环境，零侵入、高兼容、强压缩

---

## 二、核心能力

### 2.1 Token压缩

| 能力 | 说明 |
|------|------|
| 命令输出压缩 | 将长输出压缩为摘要 |
| 错误信息精炼 | 保留关键错误，删除噪音 |
| 日志截断 | 长日志只保留重要部分 |
| 表格优化 | 复杂表格转为简洁格式 |

### 2.2 技术特性

- **零侵入**: 不修改源代码
- **高兼容**: 支持所有CLI命令
- **强压缩**: 平均90%+压缩率
- **即时生效**: 无需重启

> 来源: GitHub Issue #1351

```
测试环境: Terminal-Bench 2.0

结论:
- RTK没有损害正确性
- 但改变了工具反馈，推送模型进入更嘈杂的路径
```

## 五、安装与使用

### 5.1 安装

```bash
# macOS/Linux
curl -fsSL https://rtk.ai/install.sh | sh

# Cargo
cargo install rtk
```

### 5.2 基本使用

```bash
# 直接运行命令
rtk run <your-command>

# 统计节省
rtk gain

# 查看统计
rtk stats
```

### 5.3 Claude Code 集成

```bash
# 设置RTK为Claude Code的输出处理器
rtk init

# 或者手动配置
export RTK_ENABLED=true
```

## 六、评测方法

### 6.1 评测维度

1. **Token节省率**: 原始输出 vs 压缩输出
2. **任务完成率**: 压缩后是否影响AI理解
3. **响应延迟**: 压缩带来的额外时间
4. **覆盖范围**: 支持的命令类型

### 6.2 评测命令

```bash
# 查看节省统计
rtk gain

# 详细报告
rtk stats --verbose

# 导出报告
rtk stats --export json
```
