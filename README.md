# AI编程智驾

> 项目文档已迁移至 [.claude/docs/README.md](.claude/docs/README.md)

本项目将全部配置和文档统一归集于 `.claude/` 目录下，按以下结构组织：

| 目录 | 说明 |
|------|------|
| `.claude/docs/` | 工程简介、安装指南、文档、设计素材 |
| `.claude/projects/` | 子工程（monitor、throttler、collaborator 等） |
| `.claude/knowledge/` | 知识库 |
| `.claude/agents/` | Agent 定义 |
| `.claude/skills/` | kf- 系列技能及第三方技能 |
| `.claude/helpers/` | Hook 处理器及审计脚本 |
| `.claude/templates/` | 配置模板 |

**Quick start**: `npm run monitor` 启动监测仪表盘，或见 `.claude/docs/install/` 安装指南。
