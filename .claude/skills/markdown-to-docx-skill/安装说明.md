## 技能内容

安装后，`markdown-to-docx` 技能包含以下文件：

```
markdown-to-docx/
├── SKILL.md           # 完整使用说明
├── agents/            # Agent 配置文件
├── references/        # 参考文档
├── scripts/           # 转换脚本
│   ├── convert_markdown_to_docx.sh
│   ├── render_markdown_with_dotx.sh
│   ├── template_style_filter.lua
│   ├── postprocess_template_docx.py
│   ├── validate_captions.py
│   └── render_mermaid_blocks_for_docx.py
└── LICENSE            # 许可证
```

## 主要功能

- 将 Markdown 文件转换为高质量的 Word (.docx) 文档
- 支持 Pandoc 转换引擎
- 支持 Obsidian 图片嵌入语法
- 支持 Mermaid 图表渲染
- 支持使用 Word 模板控制输出样式
- 自动修复图表注释和标题编号
- 批量转换多篇 Markdown 文件

## 使用前提

- Pandoc（用于 Markdown 转换）
- Python 3.x（用于后处理脚本）
- Node.js 和 npx（用于 Mermaid 图表渲染，可选）
- pandoc-crossref（用于交叉引用，可选）

详细使用方法请查看 `SKILL.md` 文件。
