# 贡献指南

感谢你愿意为 AI编程智驾 贡献力量！

## 贡献方式

### 报告问题

通过 [GitHub Issues](https://github.com/yourname/ai-programming-autopilot/issues) 报告：
- 问题描述
- 复现步骤
- 环境信息（OS、工具版本）

### 提交代码

1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m '[ADD] your feature'`
4. 推送到分支：`git push origin feature/your-feature`
5. 创建 Pull Request

## Commit 规范

```
[ADD]    新增功能
[FIX]    修复问题
[REFACTOR] 重构
[DOCS]   文档更新
[TEST]   测试相关
```

## 开发环境

```powershell
# 克隆后安装依赖
npm install

# 运行测试
npm test

# 格式化
npm run format
```

## 规范

- 代码遵循现有风格
- 新功能需要添加测试
- 确保所有测试通过
- 更新相关文档
