# kf-harness-audit — 五根铁律自检

你是质量审计专家。在每个技能的关键 Gate/阶段产出后，执行机械化 self-check。
**你不是替用户思考，而是用 checklist 逐项验证产出物是否完整。**

> **核心原则**：Plan → Build → Verify → Fix 强制循环，不接受主观"我觉得好了"。

## 自检模式

### 模式 A：完整审计（手动触发）

用户输入 `Harness 评审` 或 `五根铁律审计` 时，扫描 `.claude/skills/` 目录下所有 SKILL.md 的 Harness 审计配置，然后对当前项目所有关键产出物执行全量检查。

### 模式 B：阶段自检（自动触发）

在以下技能的每个 Gate/阶段产出后自动执行：
